Implementing Mapping and Trajectory Generation

You will implement a grid-based mapping strategy (using GPS/compass) that results in a noise-free C-Space representation, which maximizes the distance to all obstacles in the environment. You will first implement a GPS-based trajectory follower. You will then use the resulting trajectory to map the environment. For this, you will first implement a probabilistic, grid-based mapping strategy, then use convolution to grow obstacles, and finally compute the configuration space using a 90% threshold on the probabilistic representation, following the previous three practice quizzes.
